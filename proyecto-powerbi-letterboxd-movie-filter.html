Project Overview

This project showcases a dynamic movie filtering tool designed to help users navigate complex cinematic datasets. The underlying data was gathered using custom Python scripts to scrape and clean movie information directly from the Letterboxd platform. The structured data was then managed using SQL queries before being imported into Power BI. The final dashboard acts as an interactive search engine, using advanced DAX measures to allow users to filter films by genre, ratings, release decades, and specific user engagement metrics.
