Buenos Aires Property Price Estimation

Data Science & Real Estate Market Analysis

Project Overview

Este proyecto se centra en el análisis masivo de datos del mercado inmobiliario en la Ciudad Autónoma de Buenos Aires y el Gran Buenos Aires. El objetivo principal es desarrollar un modelo predictivo capaz de estimar el valor de mercado de las propiedades basándose en variables clave como ubicación georeferenciada, superficie y características constructivas.

Key Methodologies

  • Web Scraping: Extracción de datos de portales inmobiliarios utilizando Selenium y BeautifulSoup.
  • Data Cleaning: Procesamiento de datasets de gran escala para eliminar outliers y valores nulos.
  • Georeferencing: Uso de mapas interactivos y coordenadas para el análisis por barrios y comunas.
