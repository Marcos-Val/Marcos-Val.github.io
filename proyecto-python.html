About this project

En este proyecto analizamos cómo la elección de colores afecta la legibilidad y el impacto de los gráficos de datos. A través de librerías como Matplotlib y Pandas, exploramos diferentes paletas de colores (secuenciales, divergentes y categóricas) para demostrar las mejores prácticas a la hora de visualizar información compleja.
